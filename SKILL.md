# 虎嗅（Huxiu）风格写作 Skill

## 账号画像

**虎嗅 HuxiuApp** — 科技与商业批判性报道媒体
- 核心定位：聚焦科技与创新，为热爱思考的人提供高效信息
- 受众：互联网从业者、科技创业者、商业分析师
- 调性：**批判性 + 独立判断 + 深度洞察**
- Slogan 精神：「问题不问，心里不安」

---

## 核心写作特征

### 1. 标题公式
虎嗅标题三大模式（任选其一）：

```
【悬念转折型】
→ "一夜之间，[主体]为何[反常举动]？"
→ "[现象]，悄悄结束了"
→ "无人敢[做某事]"

【批判提问型】
→ "[热门概念]真的改变了什么？"
→ "[公司]越来越像[不像自己的东西]了"
→ "当[趋势A]遇到[趋势B]，谁来买单？"

【数据冲击型】
→ "[数字]背后，[主体]正在[悄悄发生的变化]"
→ "花了[金额]之后，[公司]终于承认了"
```

### 2. 文章结构（标准骨架）

```
【导语段】
用一个让读者感到意外的事实或问题开场
— 不从背景介绍开始，从"反常"的现象开场
— 导语要在3句话内抛出核心矛盾

【问题拆解 × 2-3节】
每节围绕一个核心问题展开
小标题可以是：
  - 疑问句：「[谁]真的[做了什么]吗？」
  - 陈述判断：「[主体]的[策略]，正在失效」
  - 反直觉结论：「[普遍认知]，恰恰是最大误区」

【批判性核心论点】
这是虎嗅区别于其他媒体的灵魂：
— 明确给出有争议的判断
— 不怕说「这件事没有外界说的那么好」
— 用数据+逻辑支撑，不是情绪发泄

【结尾】
— 不强行正能量，不下终局判断
— 留一个开放性问题让读者继续思考
— 或点出「下一个值得追问的问题是……」
```

### 3. 写作风格原则

**必须做到**：
- 每一个判断后面都有数据/案例支撑
- 对热门概念保持怀疑：AI神话？揭穿。资本故事？追问盈利逻辑
- 至少有一处「反主流判断」（与99%报道的结论不同）
- 引用具体数字（用户数、营收、增速、成本），不用「大幅增长」这类空话

**绝对不能**：
- 写公司官方通稿视角（不帮公司宣传）
- 对行业趋势无脑唱好
- 用「颠覆性」「领先行业」「引领全球」等空洞修饰词
- 结尾带明显商务广告意图

## STEP 流程

### STEP 1：确认参数
```
请告诉我：
1. 写谁/写什么事件？（公司、产品、行业现象）
2. 你认为这件事最值得追问的问题是什么？
3. 目标字数：1500 / 2500 / 3500字
```

### STEP 2：提炼「反主流判断」
在开始写作前，先确立文章的批判性核心论点：
- 主流观点是什么？
- 我们的判断和它哪里不同？
- 数据支撑是什么？

### STEP 3：全网信息采集
采集维度：
- 官方数据（财报、公告）
- 行业研报
- 离职员工/竞争对手视角（知乎、脉脉）
- 海外媒体报道（不同视角）

### STEP 4：按骨架写正文
遵循虎嗅标题公式 + 文章骨架规范

### STEP 5：自检清单
```
□ 标题是否制造了"非点进不可"的悬念或冲突？
□ 文章是否有一个明确的"与主流不同的判断"？
□ 每个判断是否有具体数据或案例支撑？
□ 结尾是否避免了强行正能量结论？
□ 全文是否出现过"颠覆性""引领"等空洞词？（出现则删除）
```
