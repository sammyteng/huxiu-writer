# 🔍 虎嗅（HuxiuApp）Writer Skill

> 让 AI 以虎嗅「批判性科技商业观察」风格写商业深度报道的可复用 Skill。

## 简介

本 Skill 基于对虎嗅数百篇深度商业科技报道的结构化解析，将虎嗅的写作特征标准化为可复用的 AI 写作指令，覆盖从问题拆解到批判性论点确立的完整流程。

## 文件结构

```

```

## 虎嗅核心写作风格

| 维度 | 特征 |
|------|------|
| 标题 | 问题式/悬念式，制造认知冲突 |
| 视角 | 批判性，不盲目跟风 |
| 结构 | 导语抛问题→多维拆解→明确判断→开放结尾 |
| 禁忌 | 公司通稿视角、空洞修饰词、强行正能量结尾 |

## 核心写作哲学

- **批判性优先**：对热门概念和成功故事保持怀疑，寻找被忽视的反面证据
- **问题式叙事**：用「为什么这件事没有说的那么好」来构建文章逻辑
- **数据克制**：每个判断必须有具体数字支撑，不用模糊形容词
- **开放结尾**：不下终局判断，抛出下一个值得追问的问题

## 如何使用

```
使用 huxiu-writer，帮我用虎嗅风格写一篇关于 [主题] 的深度分析
```

## 适合场景

✅ 对热门商业现象的批判性分析
✅ 科技公司战略质疑与深度解读
✅ 行业泡沫识别与冷静观察
✅ 反主流的独立判断类文章
